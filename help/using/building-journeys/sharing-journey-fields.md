# 여정 필드 {#sharing-journey-fields}

이 mixin은 여정 스키마에서 사용됩니다(journeyStepEvent와 관련됨). 여기에는 다음 필드가 포함되어 있습니다.

## journeyID

기본 여정의 ID입니다.

유형:string

## journeyVersionID

여정 버전의 ID입니다. 이 id는 여정의 ID를 나타냅니다.

유형:string

## 이름

여정 이름입니다.

유형:string

## 설명

여정 설명.

유형:string

## 버전

`major`.`minor`

유형:string
