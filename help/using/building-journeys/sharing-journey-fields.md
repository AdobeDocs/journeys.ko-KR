# 경로 필드 {#sharing-journey-fields}

이 혼합은 경로 스키마에서 사용됩니다(journeyStepEvent와 관련). 여기에는 다음 필드가 포함됩니다.

## journeyID

주요 여정의 ID.

유형:문자열

## journeyVersionID

경로 버전의 ID입니다. 이 id는 경로의 ID를 나타냅니다.

유형:문자열

## name

경로 이름입니다.

유형:문자열

## description

경로 설명입니다.

유형:문자열

## 버전

버전. `major``minor`

유형:문자열
