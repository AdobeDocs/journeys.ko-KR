# round {#round}

인수에 가장 근접한 정수 값을 양수 무한대로 반올림하여 반환합니다.

## 카테고리

수학

## 함수 구문

`round(<parameters>)`

## 매개 변수

* decimal
* 정수

## 서명 및 반환된 문자

`round(<decimal>)`

`round(<integer>)`

정수를 반환합니다.

## 예

`round(3.14)`

반환 3입니다.

`round(3.54)`

반환 4입니다.

`round(-3.14)`

-3을 반환합니다.

`round(3)`

반환 3입니다.
