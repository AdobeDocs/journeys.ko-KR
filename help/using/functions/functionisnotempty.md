# isNotEmpty {#isNotEmpty}

매개 변수의 문자열이 비어 있지 않으면 true를 반환합니다.

## 범주

문자열

## 함수 구문

`isNotEmpty(<parameters>)`

## 매개 변수

* 문자열

## 서명 및 반환된 문자

`isNotEmpty(<string>)`

부울을 반환합니다.

## 예제

`isNotEmpty("")`

false를 반환합니다.

`isNotEmpty("hello")`

true를 반환합니다.
