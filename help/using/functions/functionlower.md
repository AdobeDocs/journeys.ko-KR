# lower {#lower}

매개 변수의 소문자 버전을 반환합니다.

## 범주

문자열

## 함수 구문

`lower(<parameter>)`

## 매개 변수

* 문자열

## 서명 및 반환된 문자

`lower(<string>)`

문자열을 반환합니다.

## 예

`lower("A")`

&quot;a&quot;를 반환합니다.
