# upper {#upper}

매개 변수의 대문자 버전을 반환합니다.

## 범주

문자열

## 함수 구문

`upper(<parameters>)`

## 서명 및 반환된 문자

`upper(<string>)`

문자열을 반환합니다.

## 예제

`upper("b")`

&quot;B&quot;를 반환합니다.
