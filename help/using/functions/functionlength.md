# length {#length}

매개 변수에서 문자열 식의 문자 수를 반환합니다.

## 카테고리

문자열

## 함수 구문

`length(<parameters>)`

## 매개변수

* 문자열

## 서명 및 반환된 유형

`length(<string>)`

정수 반환.

## 예

`length("Hello World")`

11을 반환합니다.
