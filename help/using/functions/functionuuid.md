# uuid {#uuid}

임의 UUID(Universal Unique IDentifier)를 생성합니다.

## 카테고리

문자열

## 함수 구문

`uuid()`

## 매개 변수

이 함수에는 매개 변수가 필요하지 않습니다.

## 서명 및 반환된 유형

`uuid()`

문자열을 반환합니다.

## 예제

`uuid()`

&quot;79e70b7f-8a85-400b-97a1-9f9826121553&quot;.s를 반환합니다.
