# uuid {#uuid}

임의 UUID(Universal Unique IDentitfier)를 생성합니다.

## 범주

문자열

## 함수 구문

`uuid()`

## 매개 변수

이 함수에는 매개 변수가 필요하지 않습니다.

## 서명 및 반환된 문자

`uuid()`

문자열을 반환합니다.

## 예제

`uuid()`

&quot;79e70b7f-8a85-400b-97a1-9f982612153&quot;.s를 반환합니다.
