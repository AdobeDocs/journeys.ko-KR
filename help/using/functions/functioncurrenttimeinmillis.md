# currentTimeInMillis {#currentTimeInMillis}

현재 시간을 밀리초 단위로 반환합니다.

## 범주

날짜

## 함수 구문

`currentTimeInMillis()`

## 매개 변수

이 함수는 매개 변수를 사용하지 않습니다.

## 서명 및 반환된 문자

`currentTimeInMillis()`

정수를 반환합니다.

## 예제

`currentTimeInMillis()`

&quot;1544712617131&quot;을 반환합니다.
