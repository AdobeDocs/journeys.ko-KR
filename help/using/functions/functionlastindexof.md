# lastIndexOf {#lastIndexOf}

두 번째 매개 변수의 마지막 항목의 위치(첫 번째 인수)를 반환합니다. 일치하는 항목이 없으면 -1을 반환합니다.

## 카테고리

문자열

## 함수 구문

`lastIndexOf(<parameters>)`

## 매개 변수

| 매개 변수 | 유형 |
|-----------|------------------|
| 문자열 | 문자열 |
| 지정된 값 | 문자열 |

## 서명 및 반환된 유형

`lastIndexOf(<string>,<string>)`

정수를 반환합니다.

## 예제

`lastIndexOf("Hello, "l"))`

3을 반환합니다.

설명:

&quot;Hello&quot;에서 &quot;l&quot;의 마지막 항목은 위치 3입니다.
