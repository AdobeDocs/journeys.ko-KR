# toTimeZone {#toTimeZone}

문자열 값을 표준 시간대로 변환합니다.

## 카테고리

전환

## 함수 구문

`toTimeZone(<parameter>)`

## 매개 변수

| 매개 변수 | 설명 |
|--- |--- |
| 문자열 | 문자열 값은 영역 ID를 포함해야 합니다. 필드 참조나 표현식이 될 수 있습니다. |

## 서명 및 반환된 유형

`toTimeZone(<string>)`

시간대를 반환합니다.

## 예제

`toTimeZone("UTC")`

UTC를 반환합니다.
