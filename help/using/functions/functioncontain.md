# contain {#contain}

두 번째 인수 문자열이 첫 번째 인수 문자열에 포함되어 있는지 확인합니다.

## 범주

문자열

## 함수 구문

`contain(<parameters>)`

## 매개 변수

* 문자열

## 서명 및 반환된 문자

`contain(<string>,<string>)`

부울을 반환합니다.

## 예제

`contain("rowing is great', "great")`

true를 반환합니다.
