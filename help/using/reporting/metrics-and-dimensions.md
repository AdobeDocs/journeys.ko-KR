## 여정 차원 {#MBE_table_wk4_bnj_w2b}

아래 표는 여정 보고서에 사용된 차원 목록과 해당 정의 및 공식을 보여 줍니다.

| 차원 | 정의 |
|--- |--- |
| **작업** | 여정에 사용된 모든 작업 목록(**작업 이름 - 작업 레이블**)(예: 푸시 - 확인, 이메일 - 보상 충실도). |
| **데이터 원본** | 여정(예: Adobe Experience Platform, 예약 시스템)에서 데이터를 보강하는 데 사용되는 데이터 소스 목록(**데이터 소스 이름**). |
| **[!UICONTROL Event]** | 여정에 사용되는 모든 이벤트 목록(**이벤트 이름 - 이벤트 레이블**)(예: Geometrixx 이벤트 - Geometrixx 체크아웃). |
| **필드 그룹** | 여정의 데이터를 보강하는 데 사용되는 필드 그룹 목록(**필드 그룹 이름**)(예: 프로필 필드 그룹, Geometrixx 예약 시스템). |
| **여정** | 테스트 모드 및 라이브의 모든 여정 목록(**여정 이름**)(예: 장바구니 포기, 호텔 예약 알림). |
| **여정 버전** | 게시된 모든 여정 버전(**여정 이름 + 버전 번호**) 목록(예: 장바구니 포기 v1, 호텔 예약 알림 v2). |
| **오케스트레이션** | 여정에 정의되어 사용되는 모든 오케스트레이션 활동(**조건, 종료, 대기**)의 목록입니다. |

## 게재 차원 {#delivery-dimensions}

아래 표는 여정 보고서에 사용된 게재 차원 목록과 해당 정의 및 공식을 보여 줍니다.

| 차원 | 정의 |
|--- |--- |
| **브라우저** | 메시지를 열거나 클릭한 브라우저. |
| **배달 이름** | 배달 라벨링 및 ID입니다. |
| **장치** | 이메일/SMS/푸시 알림 열기/보기/클릭한 장치. |
| **메시지 유형** | 이메일, SMS, 푸시 알림 또는 인앱 등 게재에 사용되는 채널. |
| **모바일 앱 이름** | 모바일 애플리케이션 이름 |
| **플랫폼** | 메시지를 열고/보고/클릭한 디바이스의 플랫폼입니다. |
| **[!UICONTROL Push platform]** | 푸시 알림이 열린 장치의 플랫폼(예: iOS 또는 Android). |
| **받는 사람 도메인** | 이메일을 여는 데 사용되는 도메인. |
| **추적 URL** | 메시지에서 사용자가 클릭한 URL. |
| **추적 URL 범주** | 추적 URL에 할당된 범주. |
| **추적 URL 레이블** | 거울 페이지 등 URL에 주어지는 라벨링 문의 또는 개방하십시오. |
| **변체** | A/B 테스트 시 이메일 변형. |

## 여정 지표 {#MBE_p_p22_c4j_w2b}

아래 표는 여정 보고서에 사용된 지표 목록과 해당 정의 및 공식을 보여 줍니다.

| 지표 | 정의 |
|--- |---|
| **완료** | 정상적으로 여정이 종료된 총 개인 수입니다. |
| **완료율** | 여정에 들어온 총 개인 수와 비교하여 여정이 정상적으로 종료된 개인 수입니다. |
| **현재** | 현재 여정에 있는 총 개인 수(예: 입력한 사람 수에서 종료한 사람, 오류 및 시간 초과한 사람 제외)입니다. |
| **현재 비율** | 여정에 참가한 인원 수와 비교한 현재 여정에 있는 총 인원수. |
| **입력** | 여정에서 개별 항목을 시작하기 위해 발생한 총 이벤트 수입니다. |
| **오류** | 여정 중에 발생했지만 여정이 성공하는 것을 방지하지 못한 총 오류 수입니다. |
| **작업 중 오류** | 작업에 대해 발생한 총 오류 수입니다. |
| **데이터 보강 오류** | 데이터 소스/필드 그룹을 호출할 때 데이터 보강에 대해 발생한 총 오류 수입니다. |
| **이벤트에 오류가 있습니다** | 이벤트에 대해 발생한 총 오류 수입니다. |
| **오류율** | 여정의 총 발생 횟수와 비교한 여정 중 발생한 총 오류 수입니다. |
| **실행된 작업** | 여정에 대해 실행된 총 작업 수입니다. |
| **데이터 보강 실행됨** | 특정 필드 그룹을 가져오기 위해 데이터 소스를 호출하여 실행된 총 보강 수입니다. |
| **실행된 이벤트** | 여정에 대해 실행된 총 작업 수입니다. |
| **실행된 오케스트레이션** | 여정에 대해 실행된 총 오케스트레이션 개체 수(종료, 대기, 조건) |
| **실패** | 성공적으로 실행되지 않은 총 여정 수입니다. |
| **실패율** | 실행 여정 수와 비교하여 성공적으로 실행되지 않은 총 여정 수입니다. |

## 게재 지표 {#delivery-metrics}

아래 표는 여정에 사용된 지표 목록을 제공합니다
보고서, 보고서 정의 및 공식.

| 지표 | 정의 |
|--- |--- |
| **차단 목록에서** | 이메일을 스팸 또는 정크로 선언한 수신자 수입니다. |
| **차단 목록 비율** | 보낸 메시지와 비교하여 차단 목록에 추가하다의 총 메시지 수입니다. |
| **바운스 + 오류** | 총 전송된 총 메시지 수와 관련하여 게재 및 자동 반환 처리 시 누적된 오류의 총계입니다. |
| **바운스 + 오류율** | 보낸 메시지와 비교하여 반송된 총 메시지 수입니다. |
| **클릭** | 게재에서 콘텐츠를 클릭한 횟수입니다. |
| **클릭스루 비율** | 게재된 총 클릭 수와 게재된 메시지 수 비교. |
| **배달됨** | 보낸 총 메시지 수와 관련하여 성공적으로 보낸 메시지 수. |
| **게재율** | 성공적으로 배달된 메시지 수와 성공적으로 배달된 메시지 수입니다. |
| **오류** | 여정 중에 발생했지만 여정이 성공하는 데 방해가 되지 않은 총 오류 수입니다. |
| **하드 바운스** | 잘못된 이메일 주소와 같은 영구적인 오류의 총 수입니다. |
| **하드 바운스 비율** | 보낸 메시지와 비교하여 영구 오류로 인해 실패한 총 게재 수입니다. |
| **미러 페이지** | 미러 페이지 링크를 클릭한 수신자 수입니다. |
| **미러 페이지 속도** | 전달된 총 메시지 수와 비교한 미러 페이지 링크의 총 클릭 수입니다. |
| **열기** | 게재 중 메시지가 열린 횟수. |
| **열람율** | 게재된 메시지 수와 비교하여 열린 총 메시지 수입니다. |
| **격리** | 반송되어 주소가 격리된 메시지 수입니다. |
| **격리 비율** | 전송된 메시지와 비교한 총 격리 수입니다. |
| **거부됨** | SMTP 서버에서 스팸으로 분류된 메시지 수. |
| **거부된 비율** | 보낸 메시지와 비교하여 거부됨으로 표시된 총 메시지 수입니다. |
| **처리됨/전송됨** | 게재에 대한 총 전송 수입니다. |
| **소프트 바운스** | 전체 받은 편지함과 같은 총 임시 오류 수. |
| **소프트 바운스 비율** | 보낸 메시지와 비교한 일시적인 이유로 실패한 총 게재 수입니다. |
| **고유 클릭 수** | 게재에서 콘텐츠를 클릭한 수신자 수입니다. |
| **고유 열기 수** | 게재를 연 수신자 수. |
| **구독 취소됨** | 구독 취소 링크의 클릭 수입니다. |
| **구독 취소 비율** | 게재된 메시지와 비교한 수신자의 총 구독 취소 수입니다. |
