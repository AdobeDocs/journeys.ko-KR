# 지표 및 차원 {#concept_rfj_wpt_52b}

>[!NOTE]
>
>게재 데이터는 Adobe Campaign Standard이 있는 경우에만 채워집니다.

동적 보고서에서 사용할 수 있는 모든 구성 요소 목록과 해당 정의를 여기에서 확인할 수 있습니다.

아래 표는 여정 보고서에 사용된 차원 목록과 해당 정의를 제공합니다.

차원과 지표 간의 호환성에 대한 자세한 내용은 을 참조하십시오. [이 페이지](../assets/do-not-localize/dynamic_report_compatibility_journey.pdf).

## 여정 차원 {#MBE_table_wk4_bnj_w2b}

아래 표는 여정 보고서에 사용된 차원 목록과 해당 정의 및 공식을 보여 줍니다.

| 차원 | 정의 |
|--- |--- |
| **작업** | 모든 작업 목록(**작업 이름 - 작업 레이블**) 여정에 사용됩니다(예: 푸시 - 체크아웃 확인, 이메일 - 보상 충실도). |
| **데이터 소스** | 데이터 소스 목록(**데이터 소스 이름**)를 사용하여 여정(예: Adobe Experience Platform, 예약 시스템)에서 데이터를 보강했습니다. |
| **[!UICONTROL Event]** | 모든 이벤트 목록(**이벤트 이름 - 이벤트 레이블**)에 사용됩니다. 여정(예: Geometrixx 이벤트 - Geometrixx 체크아웃) |
| **필드 그룹** | 필드 그룹 목록(**필드 그룹 이름**)를 사용하여 여정의 데이터를 보강할 수 있습니다. 예: 프로필 필드 그룹, Geometrixx 예약 시스템 |
| **여정** | 모든 여정 목록(**여정 이름**) 테스트 모드 및 라이브(예: 장바구니 포기, 호텔 예약 알림)에서 사용할 수 있습니다. |
| **여정 버전** | 게시된 모든 여정 버전 목록(**여정 이름 + 버전 번호**) 예: 장바구니 포기 v1, 호텔 예약 알림 v2. |
| **오케스트레이션** | 모든 오케스트레이션 활동 목록(**조건, 종료, 대기**)가 정의되었으며 여정에서 사용됩니다. |

## 게재 차원 {#delivery-dimensions}

아래 표는 여정 보고서에 사용된 게재 차원 목록과 해당 정의 및 공식을 보여 줍니다.

| 차원 | 정의 |
|--- |--- |
| **브라우저** | 메시지를 열거나 클릭한 브라우저. |
| **게재 이름** | 게재 레이블 및 ID. |
| **디바이스** | 이메일/SMS/푸시 알림이 열리거나 열람/클릭한 디바이스. |
| **메시지 유형** | 이메일, SMS, 푸시 알림 또는 인앱 등 게재에 사용되는 채널. |
| **모바일 앱 이름** | 모바일 애플리케이션 이름 |
| **플랫폼** | 메시지를 열고/보고/클릭한 디바이스의 플랫폼입니다. |
| **[!UICONTROL Push platform]** | 푸시 알림이 열린 장치의 플랫폼(예: iOS 또는 Android). |
| **수신자 도메인** | 이메일을 여는 데 사용되는 도메인. |
| **추적 URL** | 메시지에서 사용자가 클릭한 URL. |
| **추적 URL 범주** | 추적 URL에 할당된 범주. |
| **추적 URL 레이블** | 미러 페이지와 같이 URL에 지정된 레이블은 당사로 연락하거나 엽니다. |
| **변형** | A/B 테스트 시 이메일 변형. |

## 여정 지표 {#MBE_p_p22_c4j_w2b}

아래 표는 여정 보고서에 사용된 지표 목록과 해당 정의 및 공식을 보여 줍니다.

| 지표 | 정의 |
|--- |---|
| **완료됨** | 정상적으로 여정이 종료된 총 개인 수입니다. |
| **완료율** | 여정에 들어온 총 개인 수와 비교하여 여정이 정상적으로 종료된 개인 수입니다. |
| **현재** | 현재 여정에 있는 총 개인 수(예: 입력한 사람 수에서 종료한 사람, 오류 및 시간 초과한 사람 제외)입니다. |
| **현재 비율** | 여정에 입력한 개인 수 대비 현재 여정에 있는 개인 수의 합계입니다. |
| **입력됨** | 여정에서 개별 항목을 시작하기 위해 발생한 총 이벤트 수입니다. |
| **오류** | 여정 중에 발생했지만 여정이 성공하는 것을 방지하지 못한 총 오류 수입니다. |
| **동작 중 오류 발생** | 작업에 대해 발생한 총 오류 수입니다. |
| **데이터 보강 오류** | 데이터 소스/필드 그룹을 호출할 때 데이터 보강에 대해 발생한 총 오류 수입니다. |
| **이벤트 오류** | 이벤트에 대해 발생한 총 오류 수입니다. |
| **오류율** | 여정의 총 발생 횟수와 비교한 여정 중 발생한 총 오류 수입니다. |
| **실행된 액션** | 여정에 대해 실행된 총 작업 수입니다. |
| **수행된 데이터 보강** | 특정 필드 그룹을 가져오기 위해 데이터 소스를 호출하여 실행된 총 보강 수입니다. |
| **실행된 이벤트** | 여정에 대해 실행된 총 작업 수입니다. |
| **실행된 오케스트레이션** | 여정에 대해 실행된 총 오케스트레이션 개체 수(종료, 대기, 조건) |
| **실패** | 성공적으로 실행되지 않은 총 여정 수입니다. |
| **실패율** | 실행 여정 수와 비교하여 성공적으로 실행되지 않은 총 여정 수입니다. |

## 게재 지표 {#delivery-metrics}

아래 표는 여정 보고서에 사용된 지표 목록과 해당 정의 및 공식을 보여 줍니다.

| 지표 | 정의 |
|--- |--- |
| **차단 목록 시** | 이메일을 스팸 또는 정크로 선언한 수신자 수입니다. |
| **차단 목록 비율** | 보낸 메시지와 비교하여 차단 목록에 추가하다의 총 메시지 수입니다. |
| **바운스 + 오류** | 총 전송된 총 메시지 수와 관련하여 게재 및 자동 반환 처리 시 누적된 오류의 총계입니다. |
| **바운스 + 오류율** | 보낸 메시지와 비교하여 반송된 총 메시지 수입니다. |
| **클릭** | 게재에서 콘텐츠를 클릭한 횟수입니다. |
| **클릭스루 비율** | 게재된 총 클릭 수와 게재된 메시지 수 비교. |
| **게재됨** | 보낸 총 메시지 수와 관련하여 성공적으로 보낸 메시지 수. |
| **게재율** | 전송된 메시지와 비교하여 성공적으로 전달된 총 메시지 수입니다. |
| **오류** | 여정 중에 발생했지만 여정이 성공하는 것을 방지하지 못한 총 오류 수입니다. |
| **하드 바운스** | 잘못된 이메일 주소와 같은 영구 오류의 총 수입니다. |
| **하드 바운스 비율** | 보낸 메시지와 비교하여 영구 오류로 인해 실패한 총 게재 수입니다. |
| **미러 페이지** | 미러 페이지 링크를 클릭한 수신자 수입니다. |
| **미러 페이지 비율** | 전달된 총 메시지 수와 비교한 미러 페이지 링크의 총 클릭 수입니다. |
| **열기** | 게재 중 메시지가 열린 횟수. |
| **열람률** | 게재된 메시지 수와 비교하여 열린 총 메시지 수입니다. |
| **격리** | 반송되어 주소가 격리된 메시지 수입니다. |
| **격리율** | 보낸 메시지와 비교한 총 격리 수입니다. |
| **거부됨** | SMTP 서버에서 스팸으로 분류된 메시지 수. |
| **거부된 비율** | 보낸 메시지와 비교하여 거부됨으로 표시된 총 메시지 수입니다. |
| **처리됨/전송됨** | 게재에 대한 총 전송 수입니다. |
| **소프트 바운스** | 전체 받은 편지함과 같은 총 임시 오류 수. |
| **소프트 바운스 비율** | 보낸 메시지와 비교한 일시적인 이유로 실패한 총 게재 수입니다. |
| **고유 클릭수** | 게재에서 콘텐츠를 클릭한 수신자 수입니다. |
| **고유 열람 수** | 게재를 연 수신자 수. |
| **가입 해지됨** | 구독 취소 링크의 클릭 수입니다. |
| **구독 취소율** | 게재된 메시지와 비교한 수신자의 총 구독 취소 수입니다. |
