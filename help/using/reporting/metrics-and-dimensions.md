# 지표 및 차원 {#concept_rfj_wpt_52b}

>[!NOTE]
>
>게재 데이터는 Adobe Campaign Standard이 있는 경우에만 채워집니다.

여기에서 동적 보고서에서 사용할 수 있는 모든 구성 요소 목록과 해당 정의를 확인할 수 있습니다.

아래 표에는 여정 보고서에 사용된 차원 목록 및 해당 정의가 나와 있습니다.

차원과 지표 간 호환성에 대한 자세한 내용은 [이 페이지](../assets/do-not-localize/dynamic_report_compatibility_journey.pdf)를 참조하십시오.

## 여정 차원 {#MBE_table_wk4_bnj_w2b}

아래 표는 여정 보고서에 사용된 차원 목록, 해당 정의 및 공식을 제공합니다.

| 차원 | 정의 |
|--- |--- |
| **작업** | 여정에 사용되는 모든 작업 목록(**작업 이름 - 작업 레이블**)입니다(예: 푸시 - 확인 체크 아웃, 이메일 - 품질 보상). |
| **데이터 소스** | 여정(예: Adobe Experience Platform, 예약 시스템)에서 데이터를 보강하는 데 사용되는 데이터 소스 목록(**데이터 소스 이름**). |
| **[!UICONTROL Event]** | 여정에 사용되는 모든 이벤트(**이벤트 이름 - 이벤트 레이블**)의 목록(예: Geometrixx 이벤트 - Geometrixx 체크 아웃). |
| **필드 그룹** | 여정에서 데이터를 보강하는 데 사용되는 필드 그룹 목록(**필드 그룹 이름**)입니다(예: 프로필 필드 그룹, Geometrixx 예약 시스템). |
| **여정** | 테스트 모드 및 라이브(예: 장바구니 포기, 호텔 예약 알림)의 모든 여정(**여정 이름**)의 목록입니다. |
| **여정 버전** | 게시된 모든 여정 버전(**여정 이름 + 버전 번호**) 목록(예: 장바구니 포기 v1, Hotel 예약 알림 v2). |
| **오케스트레이션** | 여정에 정의되고 사용되는 모든 오케스트레이션 활동(**조건, 종료, 대기**)의 목록입니다. |

## 게재 차원 {#delivery-dimensions}

아래 표에는 여정 보고서에 사용된 게재 차원 목록, 해당 정의 및 공식이 나와 있습니다.

| 차원 | 정의 |
|--- |--- |
| **브라우저** | 메시지를 열거나 클릭한 브라우저입니다. |
| **배달 이름** | 게재의 레이블 및 ID입니다. |
| **디바이스** | 이메일/SMS/푸시 알림이 열리거나 보거나 클릭한 장치입니다. |
| **메시지 유형** | 게재에 사용되는 채널(예: 이메일, SMS, 푸시 알림 또는 인앱)입니다. |
| **모바일 앱 이름** | 모바일 애플리케이션 이름 |
| **플랫폼** | 메시지가 열리거나 보거나 클릭한 장치의 플랫폼입니다. |
| **[!UICONTROL Push platform]** | 푸시 알림이 열린 장치의 플랫폼(예: iOS 또는 Android)입니다. |
| **수신자 도메인** | 이메일을 여는 데 사용되는 도메인입니다. |
| **추적 URL** | 메시지에서 사용자가 클릭한 URL입니다. |
| **추적 URL 카테고리** | 추적 URL에 지정된 카테고리입니다. |
| **추적 URL 레이블** | 미러 페이지와 같이 URL에 지정된 레이블은 Adobe에 문의하거나 엽니다. |
| **변형** | A/B 테스트의 경우 이메일의 변형입니다. |

## 여정 지표 {#MBE_p_p22_c4j_w2b}

아래 표에는 여정 보고서에 사용된 지표 목록, 그 정의 및 공식이 나와 있습니다.

| 지표 | 정의 |
|--- |---|
| **완료됨** | 일반적으로 여정을 종료한 총 개인 수입니다. |
| **완료율** | 일반적으로 여정을 종료한 총 개인 수와 여정에 입력한 총 개인 수가 비교됩니다. |
| **현재** | 현재 여정에 있는 총 개인 수(입력한 사람 - 종료한 사람, 오류 및 시간 제한)입니다. |
| **현재 비율** | 여정에 입력한 개인 수와 비교하여 현재 여정에 있는 총 개인 수입니다. |
| **입력됨** | 여정에서 개별 항목을 시작하는 데 발생한 총 이벤트 수입니다. |
| **오류** | 여정 중에 발생했지만 여정이 성공하지 못한 총 오류 수입니다. |
| **작업 오류** | 작업에 대해 발생한 총 오류 수입니다. |
| **데이터 보강 오류** | 데이터 소스/필드 그룹을 호출할 때 데이터 보강에 대해 발생한 총 오류 수입니다. |
| **이벤트 오류** | 이벤트에 대해 발생한 총 오류 수입니다. |
| **오류율** | 여정 중에 발생한 총 오류 수와 여정의 총 발생 횟수 비교 |
| **실행된 작업** | 여정에 대해 실행된 총 작업 수입니다. |
| **실행된 데이터 보강** | 특정 필드 그룹을 가져오도록 데이터 소스를 호출하여 실행된 데이터 보강 총 수입니다. |
| **실행된 이벤트** | 여정에 대해 실행된 총 작업 수입니다. |
| **실행된 오케스트레이션** | 여정에 대해 실행된 총 오케스트레이션 개체(종료, 대기, 조건) 수입니다. |
| **실패** | 성공적으로 실행되지 않은 총 여정 수입니다. |
| **실패 비율** | 실행되지 않은 총 여정 수와 실행 여정 수 비교 |

## 게재 지표 {#delivery-metrics}

아래 표는 여정에 사용된 지표 목록을 제공합니다
보고서, 정의 및 공식.

| 지표 | 정의 |
|--- |--- |
| **차단 목록 시** | 이메일을 스팸 또는 정크 메일로 선언한 받는 사람 수입니다. |
| **차단 목록 비율** | 전송된 메시지와 차단 목록에 추가하다 비교하여 총 메시지 수입니다. |
| **바운스 수 + 오류** | 총 보낸 메시지 수와 관련하여 게재 및 자동 반환 처리 중에 누적된 총 오류 수입니다. |
| **바운스 + 오류 비율** | 전송된 메시지와 비교하여 바운스된 총 메시지 수입니다. |
| **클릭** | 게재에서 콘텐츠를 클릭한 횟수입니다. |
| **클릭스루 비율** | 게재의 총 클릭 수를 게재한 메시지 수와 비교한 것입니다. |
| **게재됨** | 보낸 총 메시지 수와 관련하여 성공적으로 보낸 메시지 수입니다. |
| **배달율** | 전송된 메시지와 비교하여 성공적으로 전달된 총 메시지 수입니다. |
| **오류** | 여정 중에 발생했지만 여정이 성공하지 못한 총 오류 수입니다. |
| **하드 바운스** | 잘못된 이메일 주소와 같은 총 영구 오류 수입니다. |
| **하드 바운스 비율** | 영구 오류로 인해 실패한 총 게재 수와 전송된 메시지 수 |
| **미러 페이지** | 미러 페이지 링크를 클릭한 수신자 수입니다. |
| **미러 페이지 비율** | 미러 페이지 링크에 대한 총 클릭 수를 게재된 총 메시지와 비교한 것입니다. |
| **열기** | 게재에서 메시지를 연 횟수입니다. |
| **오픈율** | 게재된 총 메시지 수와 게재된 메시지 수를 비교한 것입니다. |
| **격리** | 바운스되어 주소가 격리된 메시지 수입니다. |
| **격리 비율** | 보낸 메시지와 비교하여 총 격리 수입니다. |
| **거부됨** | SMTP 서버로 스팸으로 분류된 메시지 수입니다. |
| **거부된 비율** | 전송된 메시지와 비교하여 거부됨으로 표시된 총 메시지 수입니다. |
| **처리/전송** | 게재에 대한 총 전송 수입니다. |
| **소프트 바운스** | 전체 받은 편지함과 같은 총 임시 오류 수입니다. |
| **소프트 바운스 비율** | 임시 이유로 인해 실패한 총 게재 수와 보낸 메시지입니다. |
| **고유 클릭 수** | 게재에서 콘텐츠를 클릭한 수신자 수입니다. |
| **고유 열기 수** | 게재를 연 수신자 수입니다. |
| **가입 해지됨** | 구독 취소 링크에 대한 클릭 수입니다. |
| **가입 해지율** | 배달된 메시지와 비교하여 수신자별 총 구독 취소 수입니다. |
